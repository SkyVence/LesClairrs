# 🎮 ProjectRed RPG

Un RPG Cyberpunk en mode terminal développé en Go.

## 🚀 Installation

```bash
git clone https://github.com/SkyVence/projet-red_rpg
cd projectred-rpg
go mod tidy
go run src/main.go
```

## 🎮 Utilisation

- `↑`/`↓` `←`/`→` : Navigation
- `Entrée` : Sélectionner
